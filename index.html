ISC Toolbox

. This is a toolbox to perform inter-subject correlation based analysis of fmri data in Matlab as well as a GUI for the visualization of the analysis results. The toolbox contains various functions to carry out the inter-subject correlation (ISC) analysis, which can be run from a GUI. A set of visualization tools - particularly designed for ISC analyses - are intergrated to the GUI. Visualization tools contain functionalities to which standard functional neuroimaging packages were difficult to adapt.
